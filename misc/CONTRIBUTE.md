# Contributing Guide

## Requirements for a resource to be added

- [x] No judgement or 'willpower' - focus on the actual indiviudal rather than how it sounds!
- [x] Preferably open-source - if not, then organisation/company/author must respect privacy (no sign-ups, etc)
- [x] Preferably free - freeing youreself from addiciton shouldn't be paywalled!

> Every resource will be investigated to be sure its correct for everyone - I am only human, so I may make mitakes!

# Pull Requests

If you are familiar with Github, you can make a Pull Request - this is the prefered way.

If you are not familiar with Github and PRs you can open a new issue and explain what you want to add there.
